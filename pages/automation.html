Detailed Curriculum

Module 1: Foundations & Cloud Flows (Power Automate)

  • Introduction to Hyperautomation and RPA concepts.
  • Building basic Cloud Flows (Triggers, Actions, Connectors).
  • Conditional Logic and Looping in Power Automate.
  • Integrating SharePoint, Outlook, and Forms.

Module 2: Robotic Process Automation (Power Automate Desktop)

  • Installation and setup of Power Automate Desktop (PAD).
  • Creating attended and unattended UI flows.
  • Handling Exceptions and Error Management (try/catch).
  • **RPA Capstone:** Automated Invoice Data Entry.

Module 3: Advanced Orchestration with n8n

  • Setting up n8n (Self-hosted vs. Cloud).
  • Nodes, Workflows, and Data Transformation (JSON/Jinja).
  • Using **Webhooks** and Custom APIs for integration.
  • Building advanced sequential and parallel workflows.

Module 4: AI & Large Language Model (LLM) Integration

  • Connecting to OpenAI/Gemini/Azure services via n8n.
  • Prompt Engineering for automated content and classification.
  • AI Builder integration in Power Automate.
  • **Advanced Capstone:** Sentiment-based Workflow Routing.
